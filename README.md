# conjur-poc

Script is intended to install CyberArk Conjur Enterprise Secrets Manager

## Requirements

1. Supported OS (listed below) with sudo capability
2. Internet connectivity

## How To Use

1. Clone this repository
2. chmod +x setup.sh
3. run ./setup.sh

## Tested Operating Systems

- CentOS 7.x
- MacOS 10.15.7
- Ubuntu 18.x
